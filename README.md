# 戒视（JieShiUI）- 数字戒断与自律成长应用

## 项目概述

戒视是一款专注于帮助用户摆脱数字沉迷、培养健康使用习惯的移动应用原型。通过科学的戒断方案、可视化的数据分析、虚拟成就系统和社区支持，激励用户保持自律并记录成长过程。

### 核心功能

- **科学戒断**：个性化的数字使用计划和逐步减量方案
- **数据可视化**：直观展示使用习惯和改进成果
- **成就系统**：勋章墙、等级提升等游戏化激励机制
- **AI助手**：智能分析和个性化建议
- **家庭互助**：家人共同参与的互助系统
- **社区支持**：同伴激励和经验分享

## 特色功能详解

### 1. 3D勋章墙

```medal-wall.html```文件实现了具有以下特点的3D成就展示系统：

- 三层环形展示区：核心勋章、分类勋章和历史勋章
- 沉浸式3D空间：使用Three.js创建蓝色数据穹顶和粒子效果
- 全息详情界面：展示3D勋章模型、成就数据和AI评语
- 勋章交互：支持360°旋转查看、悬浮动画和点击展示详情
- 无障碍适配：提供2D精简模式切换
- 新手引导：首次进入显示引导提示

### 2. 个人中心

```profile.html```文件包含以下主要模块：

- 个人资料展示：头像、等级、能量值
- 勋章墙入口：带有动态齿轮边框和粒子汇聚效果
- 勋章展示区：显示最近获得的勋章
- 设置菜单：账户安全、创作中心、通知管理、主题设置
- 家庭互助：家庭成员和互动数据
- 消息中心：通知和互动消息
- 个性化功能：主题选择、专注模式设置

### 2.1 账户与安全

```account-security.html```文件提供以下账户安全功能：

- 安全评分：直观展示账户安全状况
- 密码管理：修改密码、定期更新提醒
- 账户验证：手机号和邮箱验证
- 双重认证：提升账户安全性
- 登录设备管理：查看和移除已登录设备
- 隐私设置：个人资料和使用记录可见性控制
- 数据管理：清除历史数据选项
- 账号注销：永久删除账号的安全流程

### 2.2 创作中心

```creation-center.html```文件提供以下创作管理功能：

- 作品管理：已发布文章、视频和问答的查看、编辑与删除
- 草稿箱：未发布创作内容的保存和管理
- 评论管理：查看和回复用户评论，支持筛选和删除
- 粉丝管理：粉丝数据统计和互动管理
- 数据统计：内容浏览量、点赞数和评论数统计
- 创作工具：支持多种内容类型的创作发布

### 3. AI助手系统

```ai.html```和```ai-analysis.html```实现了智能辅助功能：

- 使用习惯分析：屏幕使用时间和模式识别
- 个性化建议：基于用户数据的改进建议
- 心理支持：情绪识别和积极引导
- 戒断计划：自适应的阶段性目标设定

### 4. 社区功能

```social.html```、```community.html```、```community-article.html```和```community-video.html```提供了社区互动功能：

- 社交圈与社区集成：从社交圈可直接访问社区页面，热角标提示新内容
- 搜索功能：支持内容搜索和快捷标签筛选
- 分类导航：推荐、经验分享、心理课堂、小组活动和工具测评等多种内容分类
- 多样化内容：文章、视频和问答等多种内容形式
- 文章详情页：支持富文本内容、关键技巧高亮和互动功能
- 视频详情页：视频播放、内容笔记和评论互动
- AI推荐区：基于用户兴趣的个性化内容推荐
- 内容创作：一键发布和多种创作类型选择
- 社区互动：点赞、评论、收藏和分享功能

### 5. 家庭互助系统

```family.html```和```family-messages.html```实现了家庭互助功能：

- 家庭成员管理：添加和管理家庭成员，支持角色设置和家庭小组创建
- 互动消息：家庭成员间的文字鼓励消息、打卡提醒和目标完成通知
- 家庭数据看板：直观展示所有家庭成员的戒断数据、活跃度和成就进度
- 家庭挑战：全家共同参与的时限挑战活动，支持创建自定义目标和奖励机制
- 家庭时间线：记录家庭成员的重要成就和里程碑，增强家庭凝聚力
- 情感支持系统：家人间可发送定制化的鼓励消息和激励语，避免简单的点赞和礼物
- 亲情提醒：基于家庭关系定制的温馨提醒系统，增强沟通效果
- 家庭成就墙：展示家庭整体的成就和进步，强化集体荣誉感

## 技术栈

- HTML5 & CSS3：前端页面结构和样式
- JavaScript (ES6+)：交互功能实现
- Three.js：3D勋章墙和视觉效果
- Chart.js：数据可视化图表
- Remix Icon：图标资源库
- WebGL：3D渲染支持
- CSS Grid & Flexbox：响应式布局

## 设计理念

应用采用蓝色为主色调，融合科技感和舒适度，通过圆角设计、微妙动画和适当的空间层次，创造专注而不失趣味的用户体验。成就系统将戒断过程游戏化，让用户在克服数字依赖的同时获得成就感和满足感。
